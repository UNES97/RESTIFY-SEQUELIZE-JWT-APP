
# RESTIFY + SEQUELIZE + JWT Authentication APP
## Installation

1 - Install dependencies 

```bash
  npm install
```
2 - Import Database
```bash
  database.sql
```
3 - Change database Config in 
```bash
  config/db.config.js
```
4 - Run the app
```bash
  node server
```
